vpn_key Cloud-Konfiguration
info So erhalten Sie den Cloud-Token:
  1. Installieren Sie Anycubic Slicer Next (Windows)
  2. Melden Sie sich mit Ihrem Anycubic-Konto an
  3. Öffnen Sie die Datei: %AppData%\AnycubicSlicerNext\AnycubicSlicerNext.conf
  4. Kopieren Sie den Wert von "access_token" (344 Zeichen)
lock Der 344-Zeichen Token aus Anycubic Slicer Next
