info So erhalten Sie den Cloud-Token:
  1. Installieren Sie Anycubic Slicer Next (Windows)
  2. Melden Sie sich mit Ihrem Anycubic-Konto an
  3. Öffnen Sie: %AppData%\AnycubicSlicerNext\AnycubicSlicerNext.conf
  4. Kopieren Sie den Wert von "access_token"
